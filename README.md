# telegram-bots

this is an telegram-bots callback server

## environments
DB_HOST - database host
DB_PORT - database port   
DB_NAME - db name   
DB_USERNAME - db user name   
DB_PASSWORD - db password   
TELEGRAM_BOT_TOKEN - telegram bot token
